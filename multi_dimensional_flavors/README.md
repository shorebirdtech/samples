# 🧊 Multi-Dimensional Flavors Sample

This is a more advanced version of the [Flavors](../flavors) sample.

It demonstrates how to configure Shorebird for Android applications that use multiple flavor dimensions (e.g., a dimension for `environment` and a dimension for `tenant`).

## Running the App

Running this app is the same as running a standard Flutter application. You can use standard `flutter run` commands with the appropriate flavor flags.
