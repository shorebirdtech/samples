# 🍦 Flavors Sample

This is the source code for the Multiple Flavors Guide. For detailed instructions, refer to:
- [Android Flavors Guide](https://docs.shorebird.dev/code-push/guides/flavors/android/)
- [iOS Flavors Guide](https://docs.shorebird.dev/code-push/guides/flavors/ios/)

It demonstrates how to configure Shorebird in a standard Flutter app that uses multiple flavors (e.g., development, staging, production).

## Running the App

Running this app is the same as running a standard Flutter application. You can use standard `flutter run` commands with the appropriate flavor flag.
